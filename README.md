# CS6650 Assignment 1 - Distributed Chat System

## Project Structure
- **/server**: Spring Boot WebSocket server.
- **/client-part2**: Multithreaded client with performance metrics.
- **/results**: CSV data and performance charts.

## Running Instructions
### Server
1. Navigate to `/server`.
2. Run `mvn clean spring-boot:run`.

### Client
1. Navigate to `/client-part2`.
2. Run `SimpleWebSocketClient.java`.
